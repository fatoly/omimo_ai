# p3-flow

Skill для Claude Code: пошаговое формирование и ведение проектной документации
по методологиям семейства **OMIMO** — [P3.express](https://omimo.org/en/modules/p3.express/)
и [micro.P3.express](https://omimo.org/en/modules/micro.p3.express/).

## Что делает

- Помогает выбрать подходящую методологию (наводящие вопросы про размер команды,
  внешние стороны, отчётность).
- Проводит онбординг: пошагово, активность за активностью, с наводящими вопросами
  на русском языке — и по ответам собирает управляющие документы проекта.
- Хранит документацию в git-репозитории пользователя (`Docs/`),
  состояние прогресса — в `Docs/.p3flow/state.yaml`, поэтому работу можно
  прервать и продолжить в новой сессии.
- Генерирует диаграммы (карта результатов, схема цикла) через отдельный skill
  [`Agents365-ai/drawio-skill`](https://github.com/Agents365-ai/drawio-skill);
  при его отсутствии деградирует до текстового представления.


## Лицензия и источник

Методологии P3.express и micro.P3.express разработаны и распространяются
[OMIMO](https://omimo.org) по лицензии **CC-BY 4.0**.

Этот skill — **неофициальная** адаптация для работы с Claude Code. Названия
активностей и документов на русском — рабочий перевод, не официальный.
При расхождениях сверяйтесь с оригинальным маналом на omimo.org.

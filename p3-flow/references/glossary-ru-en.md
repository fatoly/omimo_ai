# Глоссарий RU ↔ EN

Единый рабочий перевод терминов OMIMO. Использовать **только** эти варианты —
во всех документах, диалогах и справочниках. Коды активностей (A01, C3 …)
не переводятся.

## Документы

| RU (использовать всегда) | EN (оригинал) | НЕ использовать |
|---|---|---|
| Описание проекта | Project Description | Устав проекта, Паспорт проекта |
| Карта результатов | Deliverables Map | Карта поставок, Карта поставляемых результатов |
| Реестр сопровождения | Follow-Up Register | Журнал рисков, Реестр отслеживания |
| Реестр состояния | Health Register | Реестр здоровья, Журнал статуса |
| Общее понимание проекта | Common Understanding | Общее видение, Единое понимание |

## Роли и механики

| RU | EN | Примечание |
|---|---|---|
| Спонсор | Sponsor | |
| Менеджер проекта | Project Manager | сокращение «РМ» допустимо в диалоге |
| Ключевые участники | Key Team Members | |
| Заинтересованные стороны | Stakeholders | не «стейкхолдеры» в документах |
| Шляпа | Hat | механика micro.P3.express |
| Шляпа координатора | Coordinator hat | ≈ менеджер проекта |

| Шляпа создателя | Creator hat | техническое качество |
| Шляпа пользователя | User hat | удобство результата |
